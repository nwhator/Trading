TradingView Webhook

This project exposes a webhook at /api/webhook that accepts TradingView alerts and stores them (Supabase) and optionally forwards them.

  • API endpoint: /api/webhook
  • HMAC header: x-tv-signature (optional)
  • Plain token header: x-tv-secret (falls back)

See the README for testing steps, TradingView setup and deployment notes.
